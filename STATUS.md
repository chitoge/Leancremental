# Status Reference

This document is a compact status reference for the public surface.

Use it when you want a quick answer to three questions:

- what is implemented
- what is partial
- what is still missing

## Remaining Gaps

The main remaining work is concentrated in the more specialized parts of the API:

- Add clock `snapshot`.
- Add graph-node garbage collection for memo entries that have no live observers.
- Add inverse-based `sum` variants and optional sums (keyed aggregate updates are covered by `AssocIndexedAggregate`; unordered tree folds remain missing).
- Add `necessary_if_alive` and richer observability controls.
- Add per-node value state inspection (pass-level stats counters are already implemented via `State.stabilizeWithStats` and `State.lastPassCounters`).
- Strengthen theorems relating the `IO` engine's stable snapshots to
  `Leancremental.Pure`.
